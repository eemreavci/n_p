FestPack

This is a side project which is focused on simplifying the booking process for concerts and festivals.

Normally when someone wants to attend a concert in another city, they would need to take care of the following:

  • Finding out in which cities do your favorite artists are having concerts
  • Figure out which websites are providing concert tickets for this event.
  • Compare the prices of different ticket types and by estimating other expenses you will have during this trip, try to decide on which ticket type you would like to get.
  • Find a transportation option that suits your needs and budget.
  • Find an accommodation option that is close to the event venue and suits your needs, and budget.
  • Go through registration and payment processes of multiple websites in order to book the concert ticket, flight ticket, and hotel.

The whole process is very time and effort consuming. What FestPack does is, it knows the favorite artists of the user (which can either be imported from Spotify or added by the user), and it simplifies this process. Using FestPack users can check out different packages that were prepared for them, based on their music taste and location. These packages contain the transportation, accommodation and concert ticket. The user only sees one price for the whole package which allows them to easily judge whether it is within their budget or not.

Customization

The potential user group of this application covers almost everyone who loves live music. Thus, there is a large difference in preferences between users regarding concert tickets, hotel types, and transportation choice.


FestPack provides multiple options (bronze, silver, and gold) differing from most affordable to the highest quality in order to serve the needs of all users. It is possible to choose the tier for the concert ticket, hotel, and transportation separately which allows further customization.

Bookmarks

Attending a concert mostly requires some planning beforehand, like checking your calendar, considering your budget, or asking friends to see if anyone else is attending the same concert as you are planning. Therefore, it is not always possible to book such a package on the spot, users need some time to think. When browsing a package, it is possible to bookmark it and then the user can come back to that package from the profile page at any time.

Artists

FestPack keeps the list of the user’s favorite artists in the Artists tab. In this tab, users can search for artists and add them to their list as well. By tapping on a particular artist, it is possible to get more information about the artist and also see all the upcoming concerts from this artist as packages, which one can book immediately.

Alerts and Filters

FestPack allows users to search/filter the packages based on artist, date, or budget. Based on search criteria, users can also set up an alert and the application will notify the user when there is a new matching result.


This feature allows users to catch deals from flights, hotels or concert tickets. While providing value to the user, this feature also provides a good way to gain returning customers.

Payment and Account

FestPack provides a single point of contact for the user’s complete trip. Users get to handle the payments for concert tickets, hotels, and transportation at the same time. This saves the user time both in terms of payment processes and also regarding registration processes that are mostly required by websites providing such services.


FestPack also provides the option to sign in using a Spotify account, which further reduces the entry barrier to the application.
